Command list:

.p help - DM this message
.p help levels - Get explaination about the levels
.p blacklist - Stop receiving bot notifications
.p whitelist - Receive bot notifications again
.p freeze - Level <= 3 needed - Give the "Muted" role to everyone who has been in the server for less than a day, mute every new person.
.p unfreeze - Level <= 3 needed - Stop muting every newcomer in the server, doesn't unmute muted ones.
.p setInfoChannel - Level <= 2 needed - Sets the current channel to the info channel.
.p setWelcomeChannel - Level <=2 needed - Sets the current channel to be the welcome channel.
.p (createNation|createSecretRoom) Name Desciption Thumbnail [Message_id] - Level <= 2 needed - Add the nation/secret room to the database.
.p removeNation Name - Level <= 2 needed - Remove the nation/secret room from the database.
.p createSecretRoom Name Desciption Thumbnail Message_id - Level <= 2 needed - Add the nation to the database.
.p reload - Level 2 needed - Update the information message
.p clear [@someone] [#somewhere] - Level <= 2 needed - Deletes all messages from everyone tagged in all the channels tagged among the 5000 last messages
.p poll Text Delay - Level <= 3 needed - Create a poll on which users can answer with reactions - Text should be either 1 word or in quotes, Delay is the number of minutes before the end of the poll
.p assign Role Level <= 1 needed - Gives bot rights to one Role
.p autoassign (ENABLE|DISABLE) Role Level <= 1 - Enable/Disable autoassigning random nation when joining.
.p updateShares Role Level Level <= 3 - Manually update the share leaderboard.
.p snooze text delay - Reminds you of something in x minutes
.p ping - pings the bot
